# Folder Structure Documentation

## Overview
This documentation outlines the structure of the project directories and their purposes.

### src/app/
Everything user-facing (pages, layout, routing) lives in this directory.

### src/components/
Reusable parts (like Navbar, Footer, Buttons) are organized in this directory.

### public/
All visual files (images, videos) are stored in this directory.

### src/lib/
Helpers and utilities (like apiRequest.ts) are placed in this directory.

### src/types/
TypeScript type definitions are defined in this directory.
